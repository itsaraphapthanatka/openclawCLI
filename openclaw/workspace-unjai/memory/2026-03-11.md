async parallel_search(user_message) {

**API Call:**

#### 3. Journey Architect 3-Filter System

```python
def decide_response_type(user_message, session, search_results):
    # Filter 1: Content Depth
    is_fact_check = check_keywords(["ข้อ", "บท", "อ้างอิง"])
    is_emotional = check_keywords(["เศร้า", "เหนื่อย", "ท้อ"])
    
    # Filter 2: R-Score Threshold
    if r_score < 30: return "text_only"
    elif 30 <= r_score < 60 and has_highlights: return "video_nudge"
    elif r_score >= 60 and has_highlights: return "video_package"
    
    # Filter 3: Interest Match (มี video ตรงกับคำถาม)
    return "text_only"
```
